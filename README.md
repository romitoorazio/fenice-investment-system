# Fenice Investment System

Dashboard Next.js per il progetto di investimento a 10 anni.

## Avvio locale

```bash
npm install
npm run dev
```

Aprire `http://localhost:3000`.

## Obiettivi della prima fase

- dashboard iniziale;
- monitoraggio scenario geopolitico e macroeconomico;
- classifica aziende biotech, agritech e AI;
- storico dei punteggi;
- notifiche e report automatici.

> Il sistema è uno strumento di supporto decisionale e non esegue ordini automatici.
